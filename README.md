## MinMaxClose Hub (Firefox Extension)

3 in 1 Button Hub. For window control accessibility (useful for one-line-navigation bar setups)

- Minimize Window
- Maximize/Restore
- Close Window

https://addons.mozilla.org/en-US/firefox/addon/min-max-close-hub/
